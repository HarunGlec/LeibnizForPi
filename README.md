Installation
---
Testa Shell can easily and fastly compile and install by user.

    $ make clean
	$ make serial
	$ make pi
	$ make install

Usage
---

#### **Serial Computation**
	$ ./serial numberofiteration

   ***e.g.***
   
    $ ./serial 1000
#### **Parallel Computation**
	$./pi numberofthreads numberofiteration

***e.g.***

	$./pi 4 1000
